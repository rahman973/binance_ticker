### DB URL

### Requirements

NodeJs

### How to run

clone the repo
run command in root directory `npm i`

create `.env` file using `.env.example` file as refrence

run command `npm start` or `npm run dev`

default server will start on port 3003, if `PORT` not defined in `.env` file
